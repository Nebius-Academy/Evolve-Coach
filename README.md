# Evolve Coach — Claude Code marketplace

Public marketplace for the **Coach** plugin. This repo is a **generated mirror** of
the plugin from Nebius's private monorepo (the source of truth); CI syncs it. Don't
edit it here — changes are overwritten on the next sync.

## Install (Claude Code CLI / IDE)

```sh
/plugin marketplace add Nebius-Academy/Evolve-Coach
/plugin install coach@evolve-coach
```

Org-managed setup (managed settings, Desktop) is documented with the plugin.
